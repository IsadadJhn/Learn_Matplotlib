# Learn_Matplotlib
Repository ini berisi dokumentasi dan kumpulan kode pembelajaran saya dalam memahami library Matplotlib di Python.
Materi yang dipelajari meliputi dasar-dasar plotting, pengaturan line dan markers, penggunaan labels dan grid,pemb
uatan subplot, serta berbagai jenis visualisasi data seperti scatter plot, bar chart, histogram, dan pie chart.

Repository ini bertujuan sebagai arsip pembelajaran sekaligus referensi pribadi dalam menguasai visualisasi data menggunakan Matplotlib.
